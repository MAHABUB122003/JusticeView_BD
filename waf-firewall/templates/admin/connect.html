Client Library

Lightweight middleware for your app

Recommended
1
Register your website

Go to Clients and add your domain with origin server IP.

2
Copy your API key

Each site gets a unique key. Copy it from the clients list.

3
Add the middleware

Install the client library and add one line to your app.

4
You're protected

All requests are analyzed in real-time. Blocked attacks return a block page.

JavaScript 
const waf = require('./client/node/waf');

// Add before your routes
app.use(waf({
  apiKey: 'YOUR_API_KEY',
  server: 'http://localhost:5000'  // WAF server URL
}));
Python / Flask 
from client.python.waf import waf_middleware

# Wrap your Flask app
app.wsgi_app = waf_middleware(
  app.wsgi_app,
  api_key='YOUR_API_KEY',
  server='http://localhost:5000'
)
Python / Django 
# settings.py
WAF_API_KEY = 'YOUR_API_KEY'
WAF_SERVER = 'http://localhost:5000'

MIDDLEWARE = [
  'client.python.waf.DjangoWAFMiddleware',  # Add at top
  # ... your other middleware
]
PHP 
<?php
require_once 'client/php/waf.php';

$waf = new SecureWAF('YOUR_API_KEY');
$waf->protect();  // Add at top of your PHP file

Reverse Proxy

Zero-code: WAF sits in front of your app

Zero Config
1
Edit config.json

Set your backend URL and website name in the config file.

2
Start the proxy

Run waf_proxy.py — it sits between visitors and your app.

3
Point your domain

Update DNS or nginx to route traffic through the WAF proxy port.

config.json 
{
  "website_name": "mywebsite.com",
  "backend_url": "http://localhost:8000",
  "proxy_port": 3000,
  "backend_timeout": 30,
  "whitelist_localhost": false,
  "frontend": {
    "mode": "static",
    "static_dir": "../frontend/dist",
    "spa": true
  },
  "backend_api": {
    "prefix": "api",
    "forward": true
  }
}
Static files (JS, CSS, images) bypass WAF checks for zero latency. Only dynamic requests are analyzed.

API Reference

Base: http://localhost:5000
POST /api/analyze Analyze an incoming request for threats
Request Headers
Authorization: Bearer YOUR_API_KEY
Content-Type: application/json
Request Body
{ "domain": "yourdomain.com", "request": { "url": "/page", "method": "GET", "headers": {}, "body": "", "query_params": {}, "ip": "visitor_ip" } }
Response (Allowed)
{ "status": "allowed" }
Response (Blocked)
{ "status": "blocked", "attack_type": "SQL Injection", "block_page": "<html>..." }
GET /api/stats

Get real-time WAF statistics (total requests, blocked, allowed)

POST /api/block

Manually block an IP address from accessing your site

GET /api/logs

Retrieve attack logs with timestamps, IPs, and attack types
